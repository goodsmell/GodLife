# GodLife

GodLife — 나의 하루를 시각화하는 작은 시작

# 🌟 프로젝트를 만든 이유

“갓생을 살고 싶지만, 어떻게 시작해야 할지 모르겠다.”
나 스스로 하루를 효과적으로 기록할 수 있는 도구를 만들고 싶었다.
우테코 프리코스 마지막 주차 오픈 미션으로 나 자신의 성장을 확인하기 위해 이 프로젝트를 시작했다.

# 👥 누구를 위해 만들었나요?

1. 나 자신 : 게으름의 정석인 나. 하루의 시작 시간을 꾸준히 지키고 꾸준한 운동을 하며 하루 일과와 성장을 기록하고 싶은 사람
2. 하루 관리가 어려운 사람들: 자신이 하루를 얼마나 버리고 있는지, 또는 알차게 사용했는지 알고 싶은 사람
3. 복잡한 앱 대신 가볍고 미니멀한 웹이 필요한 사람

# 🧩 이 프로그램이 해결하는 문제

내가 언제 일어나는지, 실제 루틴이 어떤지 객관적으로 확인하기 어렵다
꾸준히 기록하더라도 기록이 어디에 쌓이고 있는지 시각적 피드백이 부족하다
작은 루틴이라도 시작하는 경험을 돕는 서비스가 필요하다

# 🚀 주요 기능 (기능 구현 목록)

## 1. 목표 설정 (온보딩)

- [x] 사용자는 이름을 설정할 수 있다
- [x] 사용자는 목표 기상 시간을 설정할 수 있다.
- [x] 사용자는 목표 러닝 기준을 설정할 수 있다.

## 1. 오늘의 갓생 기록하기

- [x] 오늘 날짜를 확인할 수 있다.
- [x] 오늘 명언을 확인할 수 있다.
- [x] 사용자는 오늘의 기상 시간을 기록할 수 있다.
- [x] 사용자는 **러닝 기록(거리 또는 시간)**을 입력할 수 있다.
- [x] 사용자는 오늘의 목표 달성률을 수치(%)로 기록할 수 있다.
- [x] 사용자는 TODO 리스트를 추가/완료/삭제할 수 있다.
- [x] 사용자는 일기를 작성하고 저장할 수 있다.
- [x] 사용자는 메모를 작성하고 저장할 수 있다.
- [x] 모든 기록은 IndexedDB에 저장되며 브라우저를 종료해도 유지된다.

## 2. 기록 조회

- [x] 사용자는 날짜 선택 UI를 통해 과거의 갓생 기록을 조회할 수 있다.
- [x] 선택된 날짜의 기상시간, 러닝, 목표달성률, TODO, 일기, 메모가 화면에 표시된다.
- [x] 해당 날짜에 기록이 없을 경우 “기록 없음” 화면이 표시된다.

## 3. 분석

- [x] 사용자는 월 단위 분석 페이지를 통해 지난 한 달의 갓생 패턴을 확인할 수 있다.
      ** 분석 페이지는 다음 지표를 제공한다. **
- [x] 기상 시간 그래프
- [X] 러닝 그래프
- [ ] 월간 TODO 완료율
- [X] 월간 일기 작성률
- [ ] 분석 데이터는 IndexedDB에 저장된 기록을 기반으로 계산한다.

## 4. 설정

- [ ] 사용자는 목표 기상 시간을 설정할 수 있다.
- [ ] 사용자는 목표 러닝 거리 또는 목표 러닝 시간을 설정할 수 있다.
- [ ] 사용자는 러닝 기록 방식을 거리(km)/시간(min) 중 하나로 선택할 수 있다.
- [ ] 설정 값은 IndexedDB에 저장되며 페이지 전환 및 새로고침에도 유지된다.
